Project Description

This project is a Mashup Generator implemented using Python and Flask. It consists of two parts:

Program 1: Command Line Application

Downloads N YouTube videos of a given singer
Converts videos to audio
Trims first Y seconds from each audio
Merges all audio clips into a single mashup file
Program 2: Web Application

Provides a web interface for user input
Generates mashup using Program 1
Creates ZIP file of output
Sends the ZIP file to the user via email
